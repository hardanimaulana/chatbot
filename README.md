# chatbot

Implementation from this source https://www.twilio.com/blog/build-a-whatsapp-chatbot-with-python-flask-and-twilio

How to run:

In the 1st terminal:
1. Activate the environment
  ```
  source whatsapp-bot-venv/bin/activate
  ```
2. Executye bot.py
  ```
  python bot.py
  ```
  
In the 2nd terminal:
1. Run ngrok
  ```
  ./ngrok http 5000
  ```
